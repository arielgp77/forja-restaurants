# ADR-0003 — Stack base

## Estado
Accepted

## Decisión
- Frontend: Next.js + TypeScript + Tailwind
- Backend/API: Next.js route handlers o capa API TypeScript
- DB: PostgreSQL + Prisma
- Auth: Clerk o Auth.js
- Pagos: Stripe
- Mensajería: Twilio
- Email: proveedor separado
- Storage: S3 compatible o Cloudinary

## Razón
Balance entre velocidad, soporte ecosistema y facilidad de operación.
