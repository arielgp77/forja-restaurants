# MB01 — JOURNEYS_PRINCIPALES

## Journey 1 — Pickup
Cliente entra, ordena, paga, restaurante prepara, cliente recoge.

## Journey 2 — Curbside
Cliente ordena, paga, llega, avisa, sistema pide spot/color/modelo, staff entrega.

## Journey 3 — Admin menú
Owner/manager edita menú, publica, sitio refleja cambios.

## Journey 4 — Operación cocina
Staff ve orden, cambia estado, marca ready/completed.

## Journey 5 — Chat IA
Cliente pregunta, IA responde desde KB, ayuda a estructurar pedido, confirma checkout.
